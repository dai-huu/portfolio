# Portfolio — Ho Huu Dai

Static single-page portfolio generated from CV.

How to preview:

Open `index.html` in a browser (double-click) or run a simple HTTP server:

```powershell
# from the project folder
python -m http.server 8000
# then open http://localhost:8000
```

Replace `avatar.jpg` with your photo (same filename) to show your picture.
